# QuickSort

经典算法 快速排序，平均性能 O (nlog2n)，最坏情况效率相当于冒泡排序 O (n^2)。

# 核心思想

  1. 选取数组一个数为 mid 值，以 mid 值划分数组，mid 下标左边是小于 mid 的数组，右边则是大于 mid 的数组
  2. 然后根据 mid 划分的两个数组，递归完成以上操作
  3. 直到数组长度为 1，也就是条件为 start < end 则可以继续划分

没有形象的东西，很难懂，你可以去看图解，然后再 Debug 代码一步步思考，然后再默写几遍，基本就掌握了
