# Class 对象

Class 对象是一个非常重要的概念，有助于反射和代理。

ClassLoader 加载 .class文件 时，会把类里的一些数值常量、方法、类信息加载到内存中，这些数据为类的元数据，最终目的是为了生成 Class对象 来描述类，这个对象会被保存到 .class文件 中， Class对象 是一个实实在在的类，用来描述该类的类，有构造方法等，但不能手动调用，由 JVM 自动创建 Class对象 ， ClassLoader 会给每个 .class文件 加载后都生成一个 Class对象 来描述该类。

创建完后，可以通过对象 .getClass() 来获取他个 Class对象 ，用来完成反射或代理等，其中有一个方法 newInstance() 用于创建对象：

  • Class.newInstance() 方法可以调用无参构造函数

  • Constructor.newInstance() 可以根据传入的参数，调用任意构造函数

这种创建对象方式和 new 对像的区别

两者创建对象的方式不同，创建条件也不同

  1. 使用 newInstance 方法只能调用无参构造，new 能调用无参和有参构造函数
  2. 使用 newInstance 时必须要保证这类已经加载并且已经建立连接，就是已经被类记载器加载完毕，而 new 不需要
  3. 前者使用的是类加载机制，是一种方法，后者 new 是创建一个新类，一种关键字

这个不能说 newInstance 不方便，相反它在反射、工厂设计模式、代理中发挥了重要作用
