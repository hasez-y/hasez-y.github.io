# Docker

Docker 是一个开源的应用容器引擎，基于谷歌的 Go 语言开发。

  • Docker 可以让开发者打包应用以及依赖包到一个轻量级、可移植的容器中，如何发布到 Linux 机器上，实现虚拟化。

  • 容器是完全使用沙箱机制，相互之间不会有任何接口

  • 更重要的是容器性能开销极低

# Docker 架构

  1. 镜像 Image

    相当于一个 root 文件系统

  2. 容器 Container

    镜像和容器的关系，就像面向对象中的类和实例一样，镜像是静态的定义，容器是镜像运行的实体；容器可以被创建、启动、删除、暂停等

  3. 仓库 Repository

    可以看成一个代码控制中心，用于保存镜像

Docker 使用客户端 - 服务器 (C/S) 架构模式，使用远程 API 来管理和创建 Docker 容器。

# CentOS 安装 Docker

Docker 默认仓库是 DockerHub，国内从 DockerHub 拉取镜像有时会遇到困难，此时可以配置镜像加速器

这里选择阿里云镜像加速 ： https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors

# Hello World

Docker 的入门 Hello World，使用 docker run 命令来在容器内运行一个应用程序

第一次运行时本地仓库并没有 ubuntu:15.10 的镜像，会自动从 DockerHub 远程仓库拉取。
