# MongoDB 基础

为什么需要 MongoDB

为了应对互联网三高而使用

  • 高并发
  • 海量数据
  • 高可用、高拓展

# 简介

MongoDB 是一个开源、高性能、无模式的 NoSql 数据库，也是最像关系型数据库的非关系型数据库

支持的数据结构非常松散，是一种类似于 JSON 格式的 BSON

下载地址: https://www.mongodb.com/try/download/community
