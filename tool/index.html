# SM.MS

一款免费的图床，图片链接支持 https，可以删除上传的图片，提供多种图片链接格式图片上传限制：每个图片最大 5M，每次最多上传 10 张。你可通过浏览器上传图片。免费存储 5GB。

官网地址：https://sm.ms

# PicGo

PicGo 是一款炫酷的图床管理工具，基于 electron-vue 开发，支持 macOS,Windows,Linux 系统，支持插件系统的图片上传和图片管理管理工具。

官网地址：https://molunerfinn.com/PicGo/
